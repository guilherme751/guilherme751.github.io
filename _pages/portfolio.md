## 🎯 Featured Projects

### FlowMIA 
This project is my Computer Science undergraduate thesis at UFES, which received the highest grade. It is a framework designed to evaluate and compare privacy leakage in different tabular generative models using Membership Inference Attacks. The framework is model-agnostic, as it only requires the synthetic data generated by the models. It was built to work with network traffic data, but it can be easily adapted to other domains. 
- **Technologies**: PyTorch, Generative Adversarial Networks (GANs), Transformers, GPTs, Scikit-learn
- **Highlights**: Developed membership inference attacks against generative models for traffic data.
- [GitHub Repository](https://github.com/guilherme751/FlowMIA)

### Secure PKI Infrastructure and HTTPS Server
Implementation of a complete Public Key Infrastructure (PKI) with a Root CA and Intermediate CA to issue TLS certificates for a secure HTTPS server. The project demonstrates both programmatic certificate generation using Python and manual PKI construction using OpenSSL, replicating real-world certificate hierarchies used in corporate environments.
- **Technologies**: Python, OpenSSL, Docker, Nginx, Linux 
- **Highlights**: Implemented a hierarchical PKI architecture with Root CA → Intermediate CA → Server certificate. Demonstrated two approaches to PKI implementation: automated (Python) and manual (OpenSSL).
- [GitHub Repository](https://github.com/guilherme751/PKI-Infrastructure-Secure-HTTPS-Server-Implementation)

### Computational Graph & Automatic Differentiation Engine
From-scratch implementation of an automatic differentiation engine based on dynamic computational graphs, similar to PyTorch’s autograd system. The project demonstrates how gradients are computed through reverse-mode differentiation (backpropagation) and enables gradient-based optimization using a custom Tensor framework.
- **Technologies**: Python, Numpy
- **Highlights**: Implemented a custom Tensor class with gradient tracking and dynamic computational graph construction. Developed a reverse-mode automatic differentiation engine capable of computing gradients via backpropagation.
- [GitHub Repository](https://github.com/guilherme751/Computational-Graph-Automatic-Differentiation-Engine)

### Chest X-ray Classification using CNNs

The project compares three approaches: a custom CNN architecture, transfer learning with frozen layers, and full fine-tuning of pre-trained models.
- **Technologies**: Python, Matplotlib / Seaborn, GradCAM++ (explainability)
- **Highlights**: Implemented a custom Tensor class with gradient tracking and dynamic computational graph construction. Developed a reverse-mode automatic differentiation engine capable of computing gradients via backpropagation.
